  8. min: The min attribute specifies the minimum value the input field should contain.
  9. max: The max attribute specifies the maximum value the input field will accept.
  10. multiple: The multiple attribute specifies that the user is allowed to enter more than one value in the "input" element. This attribute works only with input type file and email.
